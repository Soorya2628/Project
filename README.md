About My Webpage

Welcome to Hari Electrical Shop.

Hari Electrical Shop is a front-end e-coomerce website developed using HTML, CSS, JavaScript, and Bootstrap. The main goal of this project is to make buying electrical products easy, fast, and convenient. Customers can explore a wide range of products, including lights, fans, switches, wires, electrical tools, and home accessories. The website features a simple and attractive design with product categories, search, special offers, a shopping cart, a wishlist, secure payment options, and a contact page. It is fully responsive, providing a smooth experience on both mobile phones and computers. This project demonstrates my skills in front-end web development, responsive design, and creating a user-friendly interface. My aim is to build a website that is easy to use, visually appealing, and useful for customers looking for quality electrical products at affordable prices.
